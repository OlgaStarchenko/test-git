# test-git

1. git status
2. git add [files] - добавляет файлы в промежуточную область (stage)
3. git add . - добавляет файлы в промежуточную область (stage)
4. git commit -m"comment" - запись
5. git log - просмотр записей подробно(выйти из git log - Q(англ раскладка))
6. git log --oneline - просмотр записей кратко
7. git push [rep_link] [branch_name]- записать изменения в GitHub-репозиторий
8. git remote -v - посмотреть ссылку на репозиторий (по умолчанию origin)
9. git branch - посмотреть название ветки
10. git reset [files] - удалить файл из промежуточной области stage
11. git diff - просмотреть все изменения
12. git diff [file] - просмотреть изменения в конкретном файле
13. git reset --hard - удалить все изменения безвозвратно
